# CRUD-poll-api

Poll CRUD API made with express and MongoDB


## Features

 User:
 - Authentication
 - JSON Web Token
 
 Poll:
 - Get All Poll
 - Get a especific poll
 - Create a poll
 - Update a especific poll
 - Delete a especific poll
 
 Extras:
 - Pagination
